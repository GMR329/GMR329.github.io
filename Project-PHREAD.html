Pretty Handy Readily Erudite Automatic Door (PHREAD)

Face Recognition

Door Hinge Articulate

Door Hinge Prototype

The Project:

PHREAD spawned out of my Internet of Things class. It fundamentally uses a Python website to allow communication between a back end server and two ESP32s which are WiFi enabled. One ESP32 streams video which Python uses facial recognition on. When an authorized face is recognized, another signal is sent to a second ESP32 which turns a door handle. It also sends a signal to a servo motor which is supposed to operate a hinge to open and then close the door. Unfortunately, the servo was never strong enough to fully open and close the door, but can move it some distance before stalling. Information was passed in URLs to be used by Python Flask (a micro web framework).

This project was made for people who have an arm full of groceries, or just expect that doors can one day be smart enough to open and close automatically. It has a long way to go, but was amazing to see with all communications up and successful.

At a Glance:

  • Used Python Flask micro web framework as communication backbone
  • Integrated facial recognition in Python on streamed video
  • Used Arduino IDE to program ESP32 WiFi modules to control IO of door motion
  • Designed hinge in Solidworks and cut on laser cutter
