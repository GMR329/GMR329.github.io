Charlie Driving

The Project:

The controls side of the NASA lunabotics challenge had to wait for mechanical engineers to finish work to run the physical robot. So we built a prototype robot that was great for demos. The wheels worked similar to the robot where the corners could articulate for turning. We mounted a lidar to the top and had a follow algorithm that could tell the robot to keep following a hand in front of it. We had a lot of fun building Charlie fully from scratch and giving it the ability to move around autonomously.

At a Glance:

  • Used Arduino as brain to control prototype robot
  • Used lidar to implement simple autonomous follow algorithm
